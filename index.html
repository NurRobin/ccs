cc

Claude Code tmux session manager.

Install

# Import signing key
curl -fsSL https://NurRobin.github.io/cc/pubkey.gpg \
  | sudo gpg --dearmor -o /etc/apt/keyrings/cc.gpg

# Add repository
echo "deb [arch=all signed-by=/etc/apt/keyrings/cc.gpg] https://NurRobin.github.io/cc stable main" \
  | sudo tee /etc/apt/sources.list.d/cc.list

# Install
sudo apt update && sudo apt install cc

Usage

cc new [name]        Start a Claude Code session
cc ls                List sessions
cc a [name]          Attach (fuzzy match)
cc kill [name]       Kill a session
cc schedule add      Schedule a recurring job
cc log               View cron execution log
